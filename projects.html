Internship Projects

Translate and Optimize BSDF Modeling Code

Display Lab, 3M

Background

  • Customers selling displays are more inclined to buy 3M products if there is an efficient design method
  • The Virtual Sampling team is preparing optical modeling software to support diverse optical product designs, including BSDF models
  • Bidirectional Scattering Distribution Function (BSDF) captures light transmission and reflection from surfaces under all lighting conditions, addressing consumer unpredictability

Objectives

  • Refactor and optimize BSDF tracer code for better performance and memory efficiency
  • Enable compatibility between Fortran and C# for enhanced memory efficiency
  • Introduce code maintainability through restructuring and documentation
  • Incorporate testing and debugging functions

Challenges

  • Navigating a large, undocumented codebase with unfamiliar coding languages
  • Managing Fortran and C# interoperability
  • Absence of references for similar implementations

Reduce Anti-Sparkle Using Machine Learning

Display Lab, 3M

Background

  • Customers use anti-glare due to consumer demand and its trendiness in advanced displays like OLEDs, though it may disrupt viewing comfort due to design
  • Anti-sparkle film enhances optical uniformity and clarity, supporting anti-glare in high-resolution devices

Objectives

  • Traditionally, the R&D process faces trade-offs between uniformity and axial brightness, limiting design improvements
  • Use machine learning and possibly a TensorFlow Neural Network to optimize the balance between anti-sparkling and anti-glare, while minimizing the impact on other features

Challenges

  • Identifying the appropriate machine learning technique that is both new and successful
  • Develop the program to ensure usability and optimization with future data

Kajioka, Toshiyuki & Kanai, Toshimasa & Ikegami, Koji & Kozuka, Hiromitsu. (2021). Achievement of low sparkle in anti-glare spray coatings by controlling the size of polymerized species in silica sols. Journal of Sol-Gel Science and Technology, 100. 10.1007/s10971-021-05664-1.

"Sprint Smart" - Neural Network for User Stories

The Cigna Group

Background

  • Project managers often struggle to prioritize user stories in Scrum and Agile processes
  • Current methods are time-consuming and not always accurate

Objectives

  • Develop a Neural Network to prioritize a list of user stories uploaded from a Jira file
  • Train the model on historical Cigna user stories for improved accuracy
  • Integrate the model into a user-friendly website powered by AWS
  • Incorporate parameters such as description (word embedding), story points, time to complete, and type of story

Challenges

  • Fully integrating the system with AWS and enabling user input functionality
  • Allowing user to re-upload an adjusted Jira file back to their story board in Jira
  • Building and optimizing a Neural Network from scratch
